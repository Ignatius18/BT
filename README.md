# ByteClub
**ByteClub** - Веб-приложение для продажи и просмотра комплектующих для ПК. Пользователи смогут просматривать каталог товаров, искать интересующие, добавлять понравившиеся позиции в корзину и оформлять заказ. 

Система разделена на публичную и приватную части. Публичная часть позволяет посетителям просматривать каталог товаров, искать интересующие позиции и знакомиться с их описанием. Приватная часть предоставляет зарегистрированным пользователям доступ к личному кабинету, где они могут управлять своим профилем и редактировать личные данные. Архитектура приложения построена по клиент-серверной модели: серверная часть обрабатывает запросы пользователей, управляет базой данных товаров и заказов, а клиентская часть отвечает за отображение интерфейса и взаимодействие с пользователем.

## Целевая аудитория:
+ Владельцы ПК
+ Офисы
+ Учреждения образования

## Ключевой функционал:
+ Регистрация пользователей
+ Авторизация и управление пользовательской сессией
+ Просмотр каталога комплектующих
+ Поиск и фильтрация товаров по категориям
+ Добавление товаров в корзину
+ Управление личными данными пользователя

___

## Структура приложения:
1. Главная (`/`) – публичная
2. Аутентификация (`/auth/*`) – регистрация, вход, восстановление пароля
3. Каталог (`/catalog`) – динамический список
4. Детальная страница товара (`/catalog/:id`) – динамическая
5. Личный кабинет (`/profile`) – приватный
6. Избранное (`/favorites`) – приватный
7. Поиск (`/search`) – с динамическими результатами
8. Админ-панель (`/admin`) – приватный
___

## Технологический стек
### Frontend: JavaScript + HTML + CSS
HTML и CSS используются для создания структуры и стиля страниц: заголовки, карточки товаров, меню, кнопки, формы и футер.
JavaScript отвечает за отображение динамического контента на страницах: карточки товаров, списки избранного, обновление корзины и интерактивные элементы (клики, фильтры, поиск).
Навигация между страницами осуществляется через обычные ссылки и формы, а состояние пользователя хранится в сессии или локальном хранилище. 

### Backend: Java
Java подходит для backend-разработки благодаря своей надёжности, производительности и масштабируемости. Она широко используется в корпоративных системах, имеет развитую экосистему и большое количество готовых решений. Кроме того, Java обеспечивает хорошую безопасность, кроссплатформенность и удобную работу с базами данных, что делает её отличным выбором для серверной части приложений.

### База данных: MySQL
**MySQL** — надёжная реляционная база данных с поддержкой сложных связей между таблицами.
Использование MySQL позволяет хранить информацию о пользователях, товарах, заказах и избранном, а также обеспечивает масштабируемость проекта.

### Хостинг: Byethost
**Byethost** - Удобный бесплатный хост, который предоставляет возможность работать с базой данных MySQL.

## Интерфейс 
### Страница: Главная (`/`)
**Тип доступа:** публичный

**Основные блоки:**
1. **Шапка:** логотип, меню (Главная, Каталог, Войти), аватар пользователя (если авторизован)
2. **Блок приветствия:** заголовок, краткое описание магазина, кнопка «Перейти в каталог»
3. **Секция «Популярные товары»:** 6 карточек товаров
4. **Футер:** контакты, ссылки, информация о магазине
   
**Динамический контент:** карточки популярных товаров подгружаются из базы, отображение имени пользователя при авторизации

### Страница: Аутентификация (`/auth/*`)
**Тип доступа:** публичный

**Основные блоки:**
1. **Шапка:** логотип, ссылка на главную страницу
2. **Форма регистрации:** имя пользователя, пароль, подтверждение пароля
3. **Форма входа:** поля логина и пароль, кнопка «Войти»
4. **Восстановление пароля:** поля для ввода логина и ключевого слова
5. **Ссылки переключения:** вход / регистрация / восстановление пароля
   
**Динамический контент:** форма отправляет данные на сервер для проверки, вывод ошибок или успешного входа

### Страница: Каталог (`/catalog`)
**Тип доступа:** публичный

**Основные блоки:**
1. **Шапка:** логотип, меню, поиск, иконка профиля
2. **Фильтры товаров:** категория, цена, популярность
3. **Сетка товаров:** карточки товаров (изображение, название, цена)
4. **Пагинация:** переключение страниц каталога
5. **Футер:** контакты и ссылки
   
**Динамический контент:** карточки товаров и результаты фильтрации подгружаются из базы

### Страница: Детальная страница товара (`/catalog/:id`)
**Тип доступа:** публичный

**Основные блоки:**
1. **Шапка:** логотип, меню, поиск, профиль
2. **Информация о товаре:** изображение, название, цена, описание
3. **Кнопки действий:** добавить в избранное, добавить в корзину
   
**Динамический контент:** данные товара и похожие товары подгружаются из базы

### Страница: Личный кабинет (`/profile`)
**Тип доступа:** приватный

**Основные блоки:**
1. **Шапка:** логотип, меню, аватар пользователя
2. **Информация о пользователе:** имя
3. **Редактирование профиля:** изменение данных пользователя
4. **Кнопка выхода из аккаунта**
   
**Динамический контент:** данные пользователя и история подгружаются из базы

### Страница: Избранное (`/favorites`)
**Тип доступа:** приватный

**Основные блоки:**
1. **Шапка:** логотип, меню, профиль
2. **Список избранных товаров:** карточки товаров
3. **Кнопки действий:** удалить из избранного, перейти к товару
   
**Динамический контент:** карточки товаров подгружаются из базы

### Страница: Поиск (`/search`)
**Тип доступа:** публичный

**Основные блоки:**
1. **Шапка:** логотип, меню, строка поиска
2. **Поле поиска:** ввод названия товара
3. **Результаты поиска:** список товаров
4. Сообщение об отсутствии результатов
   
**Динамический контент:** результаты поиска подгружаются из базы по введённому запросу

### Страница: Админ-панель (`/admin`)
**Тип доступа:** приватный

**Основные блоки:**
1. **Шапка:** логотип, меню администратора
2. **Список товаров:** таблица с товарами
3. **Управление товарами:** добавление, редактирование, удаление
4. **Управление пользователями**
   
**Динамический контент:** данные товаров и пользователей подгружаются из базы

## Схема данных 
### Модель User:
+ `id` — id пользователя
+ `username` — логин пользователя
+ `password_hash` — зашифрованный пароль

### Модель Product:
+ `id` — id товара
+ `name` — название товара
+ `description` — описание товара
+ `price` — цена товара
+ `category` — категория товара
+ `image_url` — ссылка на изображение товара
